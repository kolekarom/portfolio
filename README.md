# 📖 Overview

This is my personal portfolio website built using **React.js**.

The website highlights my:

- 👨‍💻 Projects
- 🚀 Skills
- 💼 Experience
- 📈 Coding Profiles
- 📫 Contact Information

The portfolio is fully responsive and designed to provide recruiters and developers with a clean overview of my work.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/kolekarom/portfolio.git
```

---

## Navigate

```bash
cd portfolio
```

---

## Install Dependencies

```bash
npm install
```

---

## Run

```bash
npm start
```

Application runs on

```
http://localhost:3000
```

---

# 📦 Build

```bash
npm run build
```
